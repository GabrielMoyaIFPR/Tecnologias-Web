Plugin: Jquery Entropizer


O que é o Plugin?

Entropizer é um plugin jQuery para criar um campo de validação de senha, a onde você pode verificar o quão segura é sua senha.

Como Funciona o Plugin?

No campo de validação de senha,você irá digitar a senha pretendida,e enquanto você digita a senha é possivel se verificar se sua senha é Muito forte (Verde),Forte (Verde claro),Moderada (Amarelo) ou Fraca (Vermelho).

Demonstração do Código e passo a passo:


1-Primeiramente você deve baixar o Diretório do plugin.

Acesse o Diretório



2-Você deve procurar a pasta Test e procurar por Index.html.

-Dessa forma (jquery-entropizer/index.html).


3-Você deverá Importar os seguintes links:

    <script type="text/javascript" src="jquery-entropizer-master/lib/jquery.js"></script>
    <script type="text/javascript" src="jquery-entropizer-master/lib/entropizer.js"></script>
    <script type="text/javascript" src="jquery-entropizer-master/dist/js/jquery-entropizer.min.js"></script>
    <script src="bootstrap/bootstrap.min.js" charset="utf-8"></script>
    <link rel="stylesheet" href="bootstrap/bootstrap.min.css">
    <script type="text/javascript">

4-Após Fazer isso vá no codigo do Index que você Abriu e copie o style dentro do Head:

 <head>
    <style>

    #meter2 .entropizer-track {
      background-color: #e8e8e8;
      border-radius: 2px;
      height: 10px;
      }
      #meter2 .entropizer-bar {
        padding-top: 2px;
        height: 10px;
        }

        </style>
        </head>

5-Você Deverá criar um Input como esse:

  <div id="top" class="container">
        <div class="page-header">
        <div class="panel-body">
        <form class="form-horizontal">
        <div class="form-group">
        <label for="password2" class="col-md-2 control-label">Senha:</label>
        <div class="col-md-10">
        <input type="password" id="password2" class="form-control" placeholder="Digite uma senha" />
        </div>
        </div>
        <div class="row">
        <div id="meter2" class="col-md-10 col-md-offset-2"></div>
        </div>
        </form>
        <hr />
        </div>
        </div>
        </div>

6-Após isso Cole o código fonte do Entropizer dentro do body do seu Html:

<script>
          (function($) {
            $('#meter2').entropizer({
              target: '#password2',
              update: function(data, ui) {
                ui.bar.css({
                  'background-color': data.color,
                  'width': data.percent + '%'
                });
              }
            });
            var max = 80;
            $('#meter3').entropizer({
              create: function(container) {
                var wrapper = $('<div>').addClass('progress progress-striped active').appendTo(container);
                var bar = $('<div>').addClass('progress-bar')
                .attr({
                  'role': 'progressbar',
                  'aria-valuemin': 0,
                  'aria-valuemax': max
                })
                .appendTo(wrapper);
                return {
                  wrapper: wrapper,
                  bar: bar
                };
              },
              update: function(data, ui) {
                ui.bar.css({
                  'width': data.percent + '%'
                })
                .attr({
                  'aria-valuenow': data.entropy
                })
                .text(data.entropy.toFixed(0));
                ui.bar[0].className = 'progress-bar progress-bar-' + data.suffix;
              },
              destroy: function(ui) {
                ui.wrapper.remove();
              }
            });
          })($);
          </script>

Após isso Seu programa deverá funcionar como no exemplo.
